# Arkanoid

Title: Arkanoid Retro-Remake
A high-performance recreation of the iconic Arkanoid arcade game, built using C++ and SFML Library.
This project focuses on smooth gameplay mechanics and efficient collision detection.

Key Features:

Core Mechanics: Responsive paddle movement, realistic ball physics, and brick destruction logic.

Collision System: Custom-built 2D collision handling for accurate trajectory shifts.

Dynamic Difficulty: Scaling speed and level complexity to enhance player engagement.

Optimized Rendering: Smooth frame rates for a seamless retro experience.
